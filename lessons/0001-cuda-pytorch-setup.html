CUDA & PyTorch Environment Setup

Initialize and verify GPU-accelerated computing on your Ubuntu RTX 4060 Ti workstation.

1 Why Start Here?

Fine-tuning Large Language Models (LLMs) requires millions of matrix multiplications per second. Doing this on a CPU is impossibly slow. To utilize your NVIDIA RTX 4060 Ti GPU, we must use CUDA (Compute Unified Device Architecture) via PyTorch.

Since your GPU has 16 GB of VRAM, it is a fantastic driver for Parameter-Efficient Fine-Tuning (PEFT) models like Llama-3-8B using QLoRA. Let's make sure your Python environment is speaking to the GPU correctly.

2 Step 1: Verify System GPU Drivers

Before installing Python libraries, check if your Ubuntu system recognizes the NVIDIA GPU and driver. Run this in your terminal:

nvidia-smi

Expected Output: You should see a table detailing your GPU name (e.g., NVIDIA GeForce RTX 4060 Ti), total VRAM (around 16380 MiB), and the current CUDA Driver Version (e.g., 13.0).

3 Step 2: Create a Conda Environment

Using a Conda environment prevents package conflicts and allows easy management of PyTorch and CUDA versions. Run the following commands in your shell:

# Create a new conda environment named 'fine-tuning' with Python 3.10
conda create -y -n fine-tuning python=3.10

# Activate the conda environment
conda activate fine-tuning

4 Step 3: Install PyTorch with CUDA 12.4

Your NVIDIA driver supports **CUDA 13.0**, but PyTorch runs on CUDA 12.1 or 12.4 runtime wheels, which are fully backward-compatible. We will install PyTorch targeting the CUDA 12.4 runtime for best compatibility with your setup:

# Install via conda
conda install -y pytorch torchvision torchaudio pytorch-cuda=12.4 -c pytorch -c nvidia

# Alternatively, you can use pip inside your active conda environment:
# pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124

5 Step 4: Run the Verification Script

Create a verification script to query PyTorch. You can save this file as scripts/verify_gpu.py:

import torch

print("=== PyTorch GPU Verification ===")
cuda_available = torch.cuda.is_available()
print(f"CUDA Available: {cuda_available}")

if cuda_available:
    device_count = torch.cuda.device_count()
    current_device = torch.cuda.current_device()
    device_name = torch.cuda.get_device_name(current_device)
    total_memory = torch.cuda.get_device_properties(current_device).total_memory
    memory_gb = total_memory / (1024 ** 3)
    
    print(f"Device Count: {device_count}")
    print(f"Current Device ID: {current_device}")
    print(f"Device Name: {device_name}")
    print(f"Total VRAM: {memory_gb:.2f} GB")
    
    # Run a quick tensor computation on the GPU
    torch.manual_seed(42)
    x = torch.rand(3, 3).cuda()
    y = torch.rand(3, 3).cuda()
    z = x @ y
    print("Test Matrix Multiplication: Successful!")
else:
    print("❌ GPU not detected. PyTorch is running in CPU mode.")

Execute the script inside your activated environment:

python scripts/verify_gpu.py
